# Screenshots

## Dashboard Overview

*Add screenshots of your application here after running it*

To capture screenshots:
1. Run the application: `shiny::runApp("app2026.R")`
2. Take screenshots of different tabs and visualizations
3. Save them in a `screenshots/` folder
4. Add them to this file using markdown:

```markdown
![Home Page](screenshots/homepage.png)
![Player Stats](screenshots/player-stats.png)
![Card Analysis](screenshots/card-analysis.png)
```

## Recommended Screenshots to Add

### 1. Home Page
- Welcome screen with project introduction
- Overview statistics

### 2. Player Dashboards (for each player)
- Battle count by arena
- Trophy progression over time
- Win/loss statistics
- Favorite cards visualization

### 3. Card Analysis
- Card usage frequency charts
- Deck composition breakdowns
- Rarity distribution

### 4. Interactive Features
- Hover tooltips on charts
- Filters and controls
- Responsive visualizations

## How to Add Screenshots

1. Create a `screenshots` folder in the project root
2. Capture screenshots from the running application
3. Name them descriptively
4. Update this file with the images
5. Commit and push to GitHub
